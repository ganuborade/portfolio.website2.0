# Ganesh Borade - Portfolio Website

Welcome to my personal portfolio website! This site showcases my journey as a Computer Science Engineering student, highlighting my skills, certifications, and projects.

## About Me

I'm Ganesh Ashok Borade, a passionate Computer Science & Engineering student from Beed. Inspired by my father's dedication as a farmer, I believe in hard work, discipline, and continuous learning. My interests lie in Cloud Computing, Big Data, Software Development, and building scalable modern systems.

## Website Structure

This is a multi-page portfolio website with the following pages:

- **Home (index.html)**: Introduction and profile picture
- **About (about.html)**: Personal background and interests
- **Projects (projects.html)**: Showcase of personal projects with GitHub links
- **Skills (skills.html)**: Categorized skills in programming, web development, cloud, etc.
- **Gallery (gallery.html)**: Image gallery
- **Contact (contact.html)**: Contact information and form link

Each page features a consistent header navigation and footer for easy navigation.

## Design Features

- **Google-Inspired Design**: Clean, minimal interface with lots of white space
- **Responsive Layout**: Optimized for all devices (mobile, tablet, desktop)
- **Modern Typography**: Uses Roboto font for readability
- **Subtle Animations**: Smooth hover effects and transitions
- **Professional Color Scheme**: Blue (#4285f4) accents on clean backgrounds

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript
- ScrollReveal.js
